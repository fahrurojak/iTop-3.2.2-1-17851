# UserRequest Test Field extension

Extension ini menambahkan field **Test** (`test_mode`) pada class `UserRequest`.

## Nilai field
- `low`
- `fast`

## Posisi di form
Field ditampilkan pada bagian **Qualification**, tepat di bawah **Urgency**.

## Aktivasi
1. Login ke iTop sebagai admin.
2. Jalankan upgrade setup (`/setup`).
3. Pilih module `userrequest-test-field` lalu lanjutkan upgrade.
4. Buka form **Create User Request** dan cek field **Test**.
